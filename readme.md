# GlobalImport S.A.

Sitio web institucional para **GlobalImport S.A.**, una empresa ficticia dedicada a la **importación y distribución** de productos electrónicos, indumentaria y artículos del hogar.

El objetivo del proyecto es transmitir **confianza y profesionalismo**, presentando los servicios de la compañía de forma clara y atractiva, y facilitando el **contacto** y el **registro** de usuarios interesados.

## Contenido del sitio

- **Página principal (index.html)**
  - Presentación de la empresa (hero con banner e información de marca).
  - Sección de **Servicios**:
    - Importación de Electrónica
    - Importación de Indumentaria y Accesorios
    - Importación de Artículos del Hogar
    - Logística y Distribución
  - Sección **Por qué elegirnos** con puntos diferenciales.
  - Sección **Contacto**.
  - Footer con información corporativa y enlaces sociales.

- **Página de Registro (registro.html)**
  - Formulario dividido en secciones para capturar:
    - Datos personales
    - Datos de contacto y acceso (incluye validación/fortaleza de contraseña en el front)
    - Dirección de entrega
    - Preferencias y aceptación de términos

## Tecnologías utilizadas

- HTML
- CSS (estilos globales y estilos específicos para el registro)
- JavaScript (interacción del menú responsive y cálculo de fortaleza de contraseña)

## Estructura del proyecto

- `index.html` — Página principal
- `registro.html` — Formulario de registro
- `style.css` — Estilos de la página principal
- `registro.css` — Estilos del formulario de registro
- `assets/images/` — Recursos visuales (logo e imágenes de las secciones)

## Cómo ejecutar

1. Abrir `index.html` en el navegador.
2. Para ver el formulario, ingresar a `registro.html`.

> Nota: este proyecto es front-end estático (sin backend). El formulario envía a `#` por defecto.
